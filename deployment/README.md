# Dokus Server Deployment

This folder contains everything you need to deploy a fully functional Dokus server on any machine.

### Prerequisites

- **None!** The installation script will install Docker if needed

### Installation

#### macOS / Linux

```bash

./dokus.sh
```

#### Windows

```cmd

```

That's it! The script will:
1. ‚úÖ Check for Docker (install if needed)
2. ‚úÖ **Interactively configure all settings** (or use secure auto-generated defaults)
3. ‚úÖ Pull the latest Dokus images from our registry
4. ‚úÖ Start all services with health checks
5. ‚úÖ Optionally configure auto-start on boot

### Interactive Setup (Recommended)

The installation script prompts for all configuration values with **smart defaults**:

**Just press Enter to accept defaults:**
- Secure random passwords auto-generated
- Sensible default usernames
- Production-ready JWT configuration

**Or enter custom values:**
- Custom database credentials

### Example Interactive Session

```

Default: dokus
Value (press Enter for default): [just press Enter]

Database password:
Default: <auto-generated secure password>
Value (press Enter for default): [just press Enter]

Redis password:
Default: <auto-generated secure password>
Value (press Enter for default): [or type: MySecureRedisPass123]
```

### Manual Configuration

The `.env` file is automatically created. You can also edit it manually:

```env
# Database
DB_USERNAME=dokus
DB_PASSWORD=auto-generated-or-custom

# Redis
REDIS_PASSWORD=auto-generated-or-custom

# RabbitMQ
RABBITMQ_USERNAME=dokus
RABBITMQ_PASSWORD=auto-generated-or-custom

# JWT Authentication
JWT_SECRET=auto-generated-64-char-secret
JWT_ISSUER=https://dokus.local
JWT_AUDIENCE=dokus-api

# Logging
LOG_LEVEL=INFO
```

### Reconfiguration

To change settings after initial setup:

```bash
# Delete .env and run script again
rm .env
./dokus.sh    # or dokus.bat on Windows

# Script will offer to reconfigure
```

## üîê Security Notes

### For Production Deployment

1. **Use Strong Passwords**: Generate secure random passwords for all services
2. **Configure Firewall**: Only expose necessary ports
3. **Enable HTTPS**: Use a reverse proxy (nginx/traefik) with SSL certificates
4. **Regular Backups**: Backup PostgreSQL databases and volumes
5. **Update Regularly**: Pull latest images weekly for security updates

### Generating Secure Passwords

```bash
# Generate a secure password (32 characters)
openssl rand -base64 32

# Generate JWT secret (64 characters)
openssl rand -base64 64
```

## üîÑ Auto-Start Configuration

The installation script can configure Dokus to start automatically:

- **macOS**: Creates a LaunchAgent
- **Linux**: Creates a systemd service
- **Windows**: Uses Docker Desktop's restart policy

### Disabling Auto-Start

**macOS:**
```bash
launchctl unload ~/Library/LaunchAgents/com.dokus.server.plist
```

**Linux:**
```bash
sudo systemctl disable dokus.service
```

**Windows:**
Docker Desktop Settings ‚Üí General ‚Üí Uncheck "Start Docker Desktop when you log in"

## üÜò Troubleshooting

### Services Won't Start

1. Check Docker is running: `docker info`
2. Check logs: `docker compose logs`
3. Verify .env configuration
4. Ensure ports aren't already in use

### Can't Pull Images

1. Verify internet connection
2. Check Docker registry configuration (insecure-registries)
3. Try manually: `docker pull 94.111.226.82:5000/dokus-auth:latest`

### Database Connection Errors

1. Wait longer for services to start (can take 2-3 minutes)
2. Check database containers: `docker compose ps | grep postgres`
3. Restart services: `docker compose restart`

### Permission Denied (Linux)

```bash
# Add user to docker group
sudo usermod -aG docker $USER

# Log out and log back in
```
